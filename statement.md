Problem Statement

  Students and small teams need a lightweight, easy-to-use system to track tasks without complex software or database setups. Many tasks are forgotten or mismanaged due to lack of a simple tracking tool.

Scope of the Project

  Multi-user task management system

  File-based data storage for tasks and users

  Console-based interface for simplicity

  Basic reporting and export functionality

  Modular design for maintainability

Target Users

  Students

  Interns

  Small teams needing simple task tracking

High-Level Features

  User registration and login

  Task creation, viewing, updating status, and deletion

  Reporting: per-user task summary

  Export reports to text files

  Input validation and error handling for reliability

  Modular and easy-to-maintain code structure
