# Simple Task Manager


A beginner-friendly Java console application that helps users manage tasks.


## Features
- Add a task
- View tasks
- Delete a task


## How to Run
1. Install Java.
2. Open terminal in the project folder.
3. Run:
javac TaskManager.java
java TaskManager
